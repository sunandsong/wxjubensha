设定与机制的三处咬合

  • 「卧底不知道自己是卧底」= 变形怪的诅咒：西方怪物志里 Doppelgänger 模仿他人过深会忘记本体——你们最特别的规则得到了世界观解释，文案顺手就有了：「众怪之中混入了一只变形怪…连它自己都不知道」。
  • 长按看词 = 按住封印：现有 bindlongpress/touchend 交互一行不改，只是把「🔒长按看词」的遮罩换成会呼吸的符文封印，松手符文合拢。
  • 揭晓 = 现形咒：reveal 数据一到，卧底画像播一段「扭曲熔化→露出👾真身」的 CSS 动画；平民词/卧底词改叫「众怪的词条 / 变形怪的词条」。

与 pages/spy 真实流程逐屏对应

  • lobby：标题区换「怪物集会」+ 变形怪设定文案；「1/2 个卧底」→「混入 1/2 只变形怪」。
  • waiting：玩家列表 → 图鉴画像框（头像放进框里，没头像用怪物 emoji 顶）；ready → 觉醒（符文圈+边框亮金）；怪物种族按入座顺序从预置表轮流分配，纯前端装饰不进数据库。
  • playing：词卡 → 羊皮纸词条页 + 符文封印（交互不变）；提示语换图鉴口吻。
  • finished：reveal → 现形咒动画 + 三行词条对照；「你就是卧底 🎭」→「你就是变形怪 👾」，「你是平民 🙂」→「你是正经怪物 🦅」。

落地

  • 只动 spy 页的 wxml/wxss + 文案；发词/准备/揭晓逻辑零改动。觉醒圈/封印呼吸/现形熔化全是 CSS。
  • emoji 版（现在这个 mockup 的样子）零素材即可上线，①②立绘出图后逐张替换。
  • 顺序：②封印词条（每局盯得最久）→ ③现形咒 → ①集会觉醒。
  • 词库不用动：普通词照玩，图鉴只是壳；以后可以加一套「怪物主题词包」（美人鱼/海妖、狼人/吸血鬼…）做彩蛋场。

口径提醒：对外文案用「怪物图鉴/找不同」，避免「杀」字；「现形咒」等词不涉宗教迷信实指，属奇幻游戏通用语，备案口径安全。
