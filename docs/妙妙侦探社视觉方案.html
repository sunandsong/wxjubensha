简洁三原则（防走神设计）

  • 读字区素颜：剧情卡 = 玻璃卡 + 14.5px 大字 + 2.05 倍行距，唯一强调是金黄划线（复用现有 hl 分段数据）。没有背景图、没有动画、没有花边。
  • 花活只在过场：拉幕布（读前）、扭蛋（搜证动作）、聚光灯+烟花（结尾高潮）——都发生在「不需要读字」的 2 秒里，看完即散场。
  • 一屏一个渐变按钮（hub 同款紫粉），其余按钮玻璃底灰字；装饰限定在顶部 44px 夜空线内，永不与正文抢注意力。

与小程序的色彩对齐清单

  • 底色 #2b2436、玻璃卡 rgba(255,255,255,0.06) + 0.14 描边 —— 与 hub / 资料页完全一致。
  • 主按钮 #8b5cff → #c44dff（hub「快速加入」同款）；次强调粉 #ff5c8a、高亮金 #ffce54（数字炸弹同款）。
  • 标题渐变字 白 → 粉 → 紫，与 hub「夜幕降临 / 推理开始」同一支渐变。
  • 卡通感全部来自「形状与动效」：圆角 16px+、徽章、扭蛋、聚光灯、烟花 —— 不引入新色相，只用现有五色。

落地

  • 照旧只动 wxml/wxss 包装层，room/game/result 逻辑零改动；幕布/扭蛋/聚光灯全是 CSS transition。
  • 顺序：②剧情卡+扭蛋（收益最大）→ ③聚光灯+结案 → ①候场徽章。
  • 因为配色同源，这版可以直接当唯一主题用，不需要按剧本换肤；以后想做恐怖本专属再叠案卷皮。

口径提醒：对外文案继续走「推理/侦探社」，不出现敏感词，与备案口径一致。
