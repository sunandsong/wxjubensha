## 2. 930咖啡馆 当前设计（已成型）

### 设定
- 一家**正常营业**的咖啡馆（白天卖咖啡，也卖蛋糕），不再是「只在夜里开门」。
- 店名「930咖啡馆」（招牌名）。命案发生在**打烊后的深夜**。
- 老板**张涛**：一手蛋糕远近闻名的**甜点师**；爱在打烊后夜深人静时独自留店**研发新甜品**（顺带交代他案发当晚为何独自在店）。可为人精明阴狠、被最信的人坑垮后谁都不信、攥着所有人把柄、街坊都怕他。
- 半个月前查出**癌症晚期**，深夜在写一封没人知道内容的信。

### 三个嫌疑人（**已全部去性别化**，男女都能演，`gender:''`）
- **小美**（店员，**真凶**）：乡下来、供弟弟学费；挪用收银箱被攥把柄、被「报警」威胁。乡下懂苦杏仁土法毒。
- **阿强**（供货商，无辜）：做假账虚报进货价，被逼下跪录音；货仓有氰化物（看着像凶器，烟雾弹）；案后慌乱删录音 → 最像凶手。打烊后在吧台边、自己冲过咖啡（碰得到杯）。
- **老李**（常客/二十年老酒友，无辜）：欠债被当众羞辱、威胁短信；今晚带高利贷凑的现金来还；中途去厕所路过吧台空杯（碰得到杯）。

### 诡计 + 反转
- 小美用老家苦杏仁磨的**土法毒**，**冲咖啡之前抹在老板专用杯底**，再当众冲咖啡、躲后厨洗杯做不在场。「当众冲、躲后厨」正是障眼法。
- **反转（那封信）**：抽屉里翻出癌症诊断书 + 张涛亲笔信——他撕了老李欠条、免了阿强账、要把店留给小美。他叫三人来本是想念这封信、放过所有人。

### 3 幕结构
| 幕 | 玩家私密剧情 | 主持人手里的线索 |
| --- | --- | --- |
| 一·密室惊变 | 我是谁+背景+和老板关系+今晚为何留下+打烊后行踪 | t1 咖啡杯 |
| 二·各自的账 | 总结别人第一幕 + 自己这本账（收银箱/对账单/货仓/欠债/短信/台账）| t2 对账单、t3 货仓、t4 现金+短信、t5 台账 |
| 三·那只杯子（终幕）| 总结别人第二幕 + 小美苦杏仁+洗杯子、阿强删录音+吧台冲咖啡、老李杯子来历+去厕所路过 + 法医结果 | t6 专用杯·毒在杯底、t7 毒的来路、t8 删录音、t9 后厨水声、t11 谁碰得到空杯 + **投票** |

**分幕时序铁律**：每一幕的剧情，是**上一幕活动结束之后**才展开的（法医结果在第三幕剧情里才出现，不提前）。

## 3. 本会话做的机制 / 玩法改造（重点）

- **4 幕 → 3 幕**：合并原「层层假象 + 那只杯子」为终幕「那只杯子」（含投票）。
- **线索只归主持人**：玩家端只看自己剧情，看不到线索；主持人保存/截图后发到微信群。
  - `autoClues` 本子的线索卡加了 `&& isHost`；只展示**当前这一幕**的新线索（不累积）。
  - 线索/信预留 **`img` / `truth.letterImg`** 字段：以后把 AI 生成的逼真图放进 `assets/clues/`，填路径即可，主持人端显示图片、可长按「保存/发送到群」。**临时阶段没图就显示文字卡，截图发群。**
- **末幕不直接点凶**：去掉「杯子只归小美」式直给；新增 **t11「谁碰得到那只空杯」**——阿强冲过咖啡、老李去厕所路过、小美经手，三人都碰得到，把嫌疑摊平。真正收口靠**土法毒的来路**（只小美乡下懂；阿强是工业氰化物、老李不懂）。
  - 注：除凶手剧情 + 线索 t7 外，其他人剧情/主持人词里的「土法子」关键字已去掉，只说「不是货仓那种工业药」。
- **合并线索**：t6（毒在杯底）+ t10（专用杯）合成一条 t6「老板的专用杯·毒在杯底」；t10 已删。
- **真相页（结算页）**：
  - **真相正文所有人可见**，是一篇**小说**（`truth.text`，第一~四章；已去掉案情分析，第四章=小美作案过程的纯小说写法）。标题「930咖啡馆」。
  - **那封信（`truth.letter`）只主持人可见**，作为压轴；信内容已精简成「纯一封信」（去掉头尾叙述/引号），方便做成 AI 信件图或实物。
- **去性别化**：三角色 `gender:''`，去掉「她/姑娘/丫头/汉子/老婆/嫂子」等锁性别词；真相里小美/张涛一律具名消歧义。
- **测试身份加 玩家D**：`pages/test` 现在有 A/B/C/D 四个虚拟身份（930咖啡馆 3 角色 + 房主，凑 A房主 + B/C/D 跑满整局）。
- **隐藏另外两个本子**：`utils/scripts.js` 导出 `list: [SCRIPT_TEST]`；`byId` 兜底指向 930咖啡馆（旧 id 仍可解析）。
- **全局标题改「群本杀」**：`app.json` + `pages/index/index.json`。
- **去掉「再来一局」按钮**：结算页主持人只剩「结束游戏（解散房间）」。
