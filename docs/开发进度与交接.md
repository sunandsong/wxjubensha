## 2. 「深夜咖啡馆」当前设计（已成型）

### 故事内核
- 死者**张涛**：早年被最信的人做假账掏空家底，从此谁都不信，阴狠地把每个人的把柄攥在手里、记小本子上，往死里逼。
- 三个嫌疑人，人人都被他逼到墙角、人人有动机：
  - **小美**（店员，**真凶**）：穷家、父母厚望、供弟弟学费；挪用收银箱被攥把柄、被拿 110 吓。
  - **阿强**（供货商，无辜）：病母幼子，被逼做假账；被猫逗耗子般折磨、逼跪录音；慌乱删了录音 → 最像凶手。
  - **老李**（常客，无辜）：厂被卷款垮掉、一家老小；二十年酒友却被当众羞辱、当着儿子戳穿。
- **诡计**：小美用老家苦杏仁磨的**土法毒**，**冲咖啡之前抹在老板专用杯底**，再当众干净地冲咖啡、躲进后厨做不在场。「当众冲、躲后厨」的清白正是障眼法。
- **真相页反转**：抽屉里翻出**癌症晚期诊断书 + 遗书**——老板临死前撕欠条、免账、要把店留给小美。前面"他多坏、该死"垒得越高，遗书越扎心。

### 4 幕结构（剧情 / 线索 / 讨论三线对齐）
| 幕 | 三人剧情（第一人称，纯故事无心理戏） | 公开线索 | 疑团 |
| --- | --- | --- | --- |
| 一·密室惊变 | 各自身世/动机 | t1 咖啡杯（中毒） | 当众冲的咖啡、门反锁，**毒怎么进去的？** |
| 二·各自的账 | 另外两人的故事 | t2 对账单 / t3 阿强氰化物货仓 / t4 老李现金 / t5 台账 | 人人有动机，**是谁？**（阿强最像） |
| 三·层层假象 | 翻盘（各人视角） | t6 毒在杯底 / t7 土法毒 / t8 删录音 / t9 后厨录音 | 毒早在杯底，**"不在场"还算数吗？** |
| 四·那只杯子 | 那只杯子 + 老板过往与反常 | t10 专用杯（取放归小美） | **谁能单独碰到那只专用空杯？** → 小美 |

- 破案要拼三条：土法毒非工业（否掉阿强货仓）+ 毒在杯底（不在场作废）+ 杯子归小美 + 乡下人懂土法毒。
- 线索是**离散点状、覆盖三人**，阿强是最大烟雾弹（假账+氰化物货仓+删录音），小美最不起眼。
- 各幕第四幕剧情都用「**那张纸上写了什么，没人知道**」收尾，吊住遗书伏笔。

## 3. 这一轮做的机制改造（通用，不止咖啡馆）

- **玩家视图单页化**：去掉顶部 tab。玩家每幕只看「**第X幕**标题 + 自己第一人称故事 + 已公开线索」；世界观/公开旁白只主持人可见（主持人念给大家听）。
- **autoClues 开关**：剧本设 `autoClues:true` → 线索随幕自动公开（关闭搜证）。咖啡馆用这个；另外两个本子仍走「限次搜证」。
- **名字替换**（`makeNamer`）：剧本正文写角色名，展示时自动换成玩家昵称（死者/NPC 不换）。
- **按钮统一 loading 防抖**（`app.runOnce`）。
- **推进即时渲染**：主持人点下一幕，拿云函数返回值立即渲染，不等实时推送。
- **解散退出修复**：房主「结束游戏」删房间后，玩家无论在哪个页面/活页面还是切回重载，都能退出。
  关键点：加载用 `where({_id})` 查询（房间没了返回空数组、不 reject），对 null 统一走「房间已解散→回首页」。
