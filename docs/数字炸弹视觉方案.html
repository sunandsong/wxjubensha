加什么 / 不动什么

✓ 换皮（三样）

  • 黄橙顶光 + 轻暗角——同一手法，主题色换成炸弹的黄
  • LED 数字盒——数字放进带辉光的显示屏，防伪码变"铭牌小字"

✗ 不动

  • 功能零改动：生成、保存留证、防伪码逻辑全保持
  • 不做区间收窄、不做房间联机
  • 无音效、无粒子
