加了什么（就三样）

✓ 加法（都在现有语言内）

  • 一束粉紫顶光 + 轻暗角——底色还是 #1e1830，只是不再"平"，和 hub 鬼宅夜景的氛围接上
  • 每张卡一句旁白——"有一个人，拿到的词不一样"，字体字号不变，只换措辞
  • 词卡 → 机密卡——深紫卡面 + 粉色「机密」角标 + 按住显影，交互不变，只是把"文字框"变成"物件"

✗ 不做

  • 不换配色宇宙（民国墨绿金那版作废）
  • 不用衬线字体
  • 不加黑边/字幕卡/颗粒/动画表演
  • 不改任何布局结构和交互流程

三个界面（和现在逐像素可对照）
